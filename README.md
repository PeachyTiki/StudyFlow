# 📚 StudyFlow — University Study Tracker

A single-file offline study app for university students. Track semesters, subjects, Vorlesungen, Übungen, study time, and exam countdowns — all private, zero accounts.

---

## ✨ Features

- **📋 Today's Objectives** — Smart daily task list based on deadlines and pace, exam countdowns, semester progress bar
- **📚 Subject Manager** — Semesters as swipeable tabs (SS/WS format), subjects with Vorlesungen & Übungen counts, exam dates, deadlines
- **⏱ No-Distraction Study Mode** — Full-screen timer, tracks duration per task, logs sessions automatically
- **🤖 AI Tutor Export** — One-click export generates a prompt file for ChatGPT with subject context, task info, and teaching instructions
- **📊 Analytics** — GitHub-style study heatmap, time per subject, averages per day/session
- **📎 File Attachments** — Upload PDFs/images to individual Vorlesungen or Übungen
- **🌙 4 Visual Modes** — Dark, Light, Retro Dark, Retro Light
- **💾 Backup & Restore** — Full JSON export/import
- **🌍 Hemisphere Toggle** — Northern (SS25/WS2526) or Southern (S1/S2) semester naming

## 🚀 Quick Start

### Browser
1. Open `index.html` in any browser
2. That's it — everything runs locally

### iPhone (PWA)
1. Open in **Safari** → Share (⬆️) → **Add to Home Screen**

### Android (APK)
1. Go to [WebIntoApp.com](https://webintoapp.com)
2. Upload `apk/StudyFlow.zip`
3. Enable localStorage ✓ / Disable Pull to Refresh ✓

### GitHub Pages
```
https://YOUR_USERNAME.github.io/StudyFlow/
```
